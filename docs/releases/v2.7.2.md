# DevHaven v2.7.2 Release Notes

- 发布日期：2026-03-04
- Tag：`v2.7.2`
- 对比版本：`v2.7.1` -> `v2.7.2`

## 本次升级内容

### 1. 终端运行体验升级（对齐 IDE 心智）
- 运行配置编辑流程与详情面板对齐，终端内可直接管理运行配置。
- 运行任务承载迁移到底部 Run 面板，支持多运行 Tab 并保留执行输出与退出码，减少主终端干扰。

### 2. 终端交互入口与状态同步优化
- 终端头部的快捷命令控制进一步收敛，运行入口更直观，状态展示更一致。
- Worktree 初始化恢复链路增强：补齐交互锁释放后的兜底恢复，降低事件丢失导致的 `creating` 悬挂概率。

### 3. 输入体验改进
- 全局禁用输入自动纠错与自动首字母大写，避免命令、分支名、参数等被系统自动篡改。

## 版本升级

本次发布将应用版本统一升级为 `2.7.2`：
- `package.json`
- `package-lock.json`
- `src-tauri/tauri.conf.json`
- `src-tauri/Cargo.toml`
- `src-tauri/Cargo.lock`

## 兼容性说明

- 无新增迁移步骤。
- 本地数据目录仍为 `~/.devhaven/`。
