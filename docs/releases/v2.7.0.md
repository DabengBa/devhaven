# DevHaven v2.7.0 Release Notes

- 发布日期：2026-02-27
- Tag：`v2.7.0`
- 对比版本：`v2.6.0` -> `v2.7.0`

## 本次升级内容

### 1. 通用脚本中心升级
- 设置页新增通用脚本可视化管理器，支持清单编辑、参数配置、脚本内容直接编辑。
- 脚本清单改为 manifest 驱动，并补齐自动保存与合法性校验。
- 新增“恢复内置预设（仅补齐缺失项）”，内置 Jenkins 部署与远程日志查看脚本。

### 2. 快捷命令执行链路重构
- 快捷命令生命周期迁移到后端状态机统一管理，新增 `quick_command_start/stop/finish/list/snapshot` command。
- 前端改为快照 + 事件驱动的状态同步，减少运行态漂移和并发竞态。
- 移除“停止命令文本”配置项，停止行为统一收敛为运行态终止流程。

### 3. 终端与主界面交互整理
- 优化项目卡片/列表模式下的快捷操作布局，减少高频操作路径。
- 移除独立 Monitor 子窗口链路，Codex 运行状态保留在主界面统一展示。
- 简化终端内 Codex 状态相关实现，清理冗余前后端接线。

## 版本升级

本次发布将应用版本统一升级为 `2.7.0`：
- `package.json`
- `package-lock.json`
- `src-tauri/tauri.conf.json`
- `src-tauri/Cargo.toml`
- `src-tauri/Cargo.lock`

## 兼容性说明

- 无新增迁移步骤。
- 本地数据目录仍为 `~/.devhaven/`。
- 共享脚本默认目录继续使用 `~/.devhaven/scripts`，内置预设按“缺失补齐”策略写入。
